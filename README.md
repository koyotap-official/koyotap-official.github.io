# KoyoTap GitHub Pages Root Site

This repository publishes the KoyoTap official website:

`https://koyotap-official.github.io/`

## Files

- `index.html` - homepage
- `privacy-policy.html` - privacy policy
- `services.html` - service information
- `team.html` - team information
- `app-ads.txt` - public AdMob authorized seller declaration
- `assets/site.css` - shared styles
- `assets/koyotap-logo.svg` - logo asset
- `.nojekyll` - disables Jekyll processing

## Expected Public URLs

- Developer website: `https://koyotap-official.github.io/`
- Privacy policy: `https://koyotap-official.github.io/privacy-policy.html`
- Services: `https://koyotap-official.github.io/services.html`
- Team: `https://koyotap-official.github.io/team.html`
- AdMob app-ads.txt: `https://koyotap-official.github.io/app-ads.txt`
- Arrow Next privacy policy: `https://koyotap-official.github.io/ArrowNext/privacy-policy.html`
